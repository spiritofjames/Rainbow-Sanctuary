Pre-launch note

This policy is tailored to the current website build. The operating legal entity, address, governing law, service providers, and dedicated contact details must be confirmed in site-config.js before public data collection or paid enrollment begins.

1. Current website behavior

The current Rainbow Sanctuary build does not set first-party cookies, use browser local storage, run analytics, display advertising, create behavioral profiles, or use marketing pixels. Because there are no optional cookies to accept or reject, the site does not display a consent banner.

The production hosting provider may still process ordinary HTTP request and security-log data. That is explained in the Privacy Policy and must be confirmed once hosting is selected.

2. Google Fonts

The site currently loads typefaces through the Google Fonts Web API. This causes the browser to request stylesheet and font files from Google domains. Google states that this API is unauthenticated and does not set or log cookies. It is still a third-party network request and should be reassessed for the final hosting market; self-hosting the font files is an available privacy improvement.

3. If tools are added later

Analytics, embedded video, maps, scheduling widgets, payment tools, chat, or social-media embeds may introduce cookies or similar storage. Before any non-essential technology is activated, Rainbow Sanctuary must:

  • document the provider, purpose, data, duration, and countries involved;
  • update this policy and the Privacy Policy;
  • provide a consent control where applicable;
  • keep optional tools off until the visitor makes a valid choice where consent is required;
  • make withdrawal as easy as acceptance.

4. Browser controls

You can use browser settings to inspect, block, or delete cookies. Blocking essential storage may affect future site functions, but it does not affect the current static experience because the site does not set cookies.

5. Contact

Questions about site technologies can be sent to the privacy contact listed before launch.
